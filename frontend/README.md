# POS Coffee Shop Frontend

React 19 + TypeScript single-page application for the POS Coffee Shop system.

## Main Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Recharts

## Useful Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

Set `VITE_API_URL` when the API is not served from the same origin.
